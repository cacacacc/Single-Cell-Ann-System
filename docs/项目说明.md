# 单细胞高维向量 ANN 检索系统（详版）

本文保留 README.md 中移出的完整说明，适合需要更深入了解项目背景、协作分工与扩展规划的读者。

## 项目简介

本项目是一个面向单细胞高维向量数据的近似最近邻检索系统。

系统的核心目标是：读取单细胞数据文件，提取细胞向量表示，构建 ANN 近似最近邻索引，并通过 Web 页面提供 Top-K 相似细胞检索功能。

本项目整体定位为：

> 数据处理与算法为主，前端展示为辅。

在中期阶段，项目重点放在数据读取、向量提取、ANN 索引构建、检索接口封装和基础结果展示上。

## 项目背景

单细胞测序数据通常包含大量细胞样本。经过预处理后，每个细胞都可以表示为一个高维向量，例如基因表达向量或 PCA 降维后的向量。

当细胞数量较大、向量维度较高时，传统精确最近邻搜索的计算成本较高。近似最近邻检索（Approximate Nearest Neighbor, ANN）能够在保证较好检索效果的同时提升查询效率，因此适合用于单细胞相似细胞搜索场景。

本项目通过 FAISS、HNSWLIB 等 ANN 工具，实现对单细胞向量数据的快速 Top-K 相似检索，并已支持 FAISS-PQ。

## 核心模块说明

### data_reader.py

该模块负责读取和解析单细胞数据。

主要功能：

- 读取 `.h5ad` 文件
- 获取细胞数量
- 获取细胞向量矩阵
- 获取指定细胞的元数据
- 获取指定细胞的向量表示

### ann_indexer.py

该模块负责 ANN 索引构建和检索。

主要功能：

- 初始化 ANN 索引
- 构建索引
- 执行 Top-K 检索
- 保存索引文件
- 加载索引文件

### app.py

该模块负责启动后端 Web 服务。

主要功能：

- 初始化 Flask 应用
- 加载数据
- 加载或构建索引
- 提供 `/api/search` 接口
- 返回 JSON 检索结果

### frontend

该部分负责前端展示。

主要功能：

- 输入查询细胞编号
- 设置 Top-K 参数
- 请求后端接口
- 展示检索结果表格
- 可选展示可视化图表

vectors.shape == (cell_count, vector_dim)
vectors.dtype == np.float32
