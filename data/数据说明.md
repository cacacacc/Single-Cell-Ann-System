#### 数据字段说明


数据采用 `.h5ad` 格式存储

- `obs`：细胞（cell）层面的元数据
- `var`：基因（gene）层面的注释    
- `obsm`：降维结果（PCA/UMAP/tSNE）   
- `uns`：全局数据描述信息

#### 跨数据集联合处理

项目提供 `scripts/prepare_datasets.py` 脚本，用于将多个 `.h5ad` 数据集清洗并对齐到统一基因空间。当前两个肝脏数据集基因维度一致，均为 32397 个基因，适合作为跨数据集对比和联合索引构建的实验数据。

示例命令：

```bash
conda activate single-cell-ann

python scripts/prepare_datasets.py \
  data/liver.h5ad data/liver_IFALD.h5ad \
  --output data/joint_liver_aligned.h5ad \
  --report data/joint_liver_aligned_report.json \
  --join inner \
  --normalize-total \
  --log1p
```

其中：

- `--join inner`：保留两个数据集共有的基因，确保 feature 维度一致
- `--normalize-total`：对每个细胞做总量归一化
- `--log1p`：对表达矩阵做对数变换
- `joint_liver_aligned_report.json`：记录输入数据集、细胞数、基因数和对齐结果，便于实验报告引用
